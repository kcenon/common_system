Detailed Description

Event bus abstraction and common events.

This header defines a uniform way to publish/subscribe events across modules. When monitoring integration is enabled, it forwards to the monitoring_system implementation; otherwise it provides a no-op stub that keeps client code consistent.

Example:

auto bus = common::get_event_bus();
bus->publish(common::events::module_started_event{"network_system"});
